# SkyJourney - Modern Travel Booking Website

A responsive travel booking website built with HTML, Tailwind CSS, and JavaScript. The design focuses on air travel and exploration with a modern, clean interface.

## Features

- Responsive design that works on all devices
- Modern UI with smooth animations and transitions
- Interactive components with JavaScript
- Mobile-friendly navigation
- Beautiful hero section with background image
- Partner logos section with hover effects
- Popular destinations showcase
- Step-by-step booking process
- Promotional section with special offers

## Technologies Used

- HTML5
- Tailwind CSS (via CDN)
- JavaScript (Vanilla)
- Font Awesome Icons
- Custom CSS animations

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. No build process required as we're using Tailwind via CDN

## Project Structure
